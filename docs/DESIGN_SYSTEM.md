## 2. 타이포그래피 — 나눔스퀘어 라운드

- **폰트 패밀리:** `NanumSquareRound`, 폴백 `Pretendard, -apple-system, system-ui, sans-serif`
- **웨이트:** Regular 400 / Bold 700 / ExtraBold 800(제목·CTA)
- **세팅:** `public/fonts/NanumSquareRound-{R,B,EB}.woff2` → `styles/fonts.css`의 `@font-face`, `font-display: swap`

| 스타일 | 크기/행간 | 웨이트 | 용도 |
|---|---|---|---|
| Display | 28 / 34 | 800 | 화면 타이틀("Activity", "Saved") |
| Title | 22 / 28 | 800 | 카드 헤드라인, 코스명 |
| Body | 16 / 24 | 400 | 본문 |
| Body-strong | 16 / 24 | 700 | 강조 본문 |
| Meta | 14 / 20 | 400 | 지역·거리·시간 메타(muted) |
| Caption | 12 / 16 | 700 | 배지·라벨 |

AllTrails는 굵은 산세리프 헤드라인을 쓰므로, **제목은 ExtraBold**로 라운드 폰트의 친근함 + 무게감을 동시에.

## 3. 반경 · 간격 · 그림자

- **반경:** 카드/시트 `--r-lg: 20px`, 중간(이미지/썸네일) `--r-md: 16px`, 칩/배지 `--r-sm: 12px`, **pill 버튼 `--r-full: 9999px`**
- **간격 스케일(4px 기반):** 4 · 8 · 12 · 16 · 20 · 24 · 32. 화면 좌우 패딩 기본 `16px`
- **그림자:** 다크 테마라 그림자는 최소. 바텀시트/플로팅 컨트롤만 `0 2px 12px rgba(0,0,0,.5)` + 미묘한 상단 하이라이트
- **탭 타깃:** 최소 `44×44`(UX §5 한 손·야외)

## 4. 컴포넌트 인벤토리 (shadcn/ui 패턴)

Radix 프리미티브 + CVA 변형. AllTrails 룩으로 테마 오버라이드.

| 컴포넌트 | 변형 | 레퍼런스 |
|---|---|---|
| `Button` | `primary`(라임 pill), `secondary`(흰 pill), `ghost`, `icon`(원형 다크) | AllTrails "Map"·"Navigate" 버튼 |
| `Card` | `course`(썸네일+메타+♡), `stat`, `empty` | Explore 코스 카드 |
| `Chip` | `filter`(아웃라인, ⌄), `purpose`(🌳🛡️🤫🦵 토글), `category` | Explore 필터, 네이버 카테고리 칩 |
| `Sheet`(BottomSheet) | 드래그 핸들, snap points | 지도 하단 시트, 조건/태그 시트 |
| `Tabs` | 언더라인형(You/Community…) | Activity·Saved 상단 탭 |
| `Badge` | `difficulty`, `confidence`, `rarity` | 난이도·신뢰도 표시 |
| `RatingStars` | 표시/입력 | 코스 평점 |
| `Slider` | 시간·거리 | 조건 시트 |
| `BottomTabBar` | 5탭(아이콘+라벨, 활성 흰색) | 전 화면 공통 |
| `Toast` | 비모달 알림 | 메모 저장·핀 신고 |
| `MapControls` | 플로팅 원형 스택 | **네이버 스타일**(§6) |

**아이콘:** `lucide-react`(라인 스타일, AllTrails 근사). 목적 아이콘은 이모지(🌳🛡️🤫🦵)로 컬러 비의존(UX §5) 유지.

## 5. 하단 탭 네비게이션 (AllTrails → 산책온 매핑)

| AllTrails | 산책온 | 아이콘 | 역할 |
|---|---|---|---|
| Explore | **홈** | search/compass | 추천 진입(대표 코스 1개) |
| — | **탐험** | map | 지도·대전 탐험 도감 |
| Navigate | (산책 루프) | navigation | 추천→상세→산책(스택, 탭 아님) |
| Saved | **저장** | heart | 저장 코스/커스텀 루트 |
| Activity | **기록** | activity | 히스토리·연속기록·회고 |
| Profile | **프로필** | user | 계정·설정 |

> 5탭 확정: **홈 · 탐험 · 기록 · 저장 · 프로필**. 산책 루프(추천→상세→산책→완료→회고)는 홈에서 여는 **스택 플로우**(하단 탭 아님).

## 6. 지도 페이지 chrome — 네이버지도 스타일 (`navermap.jpg` 기준)

TMAP가 타일을 렌더하고, 그 위에 **네이버 스타일 UI**를 오버레이한다.

- **상단 검색 바:** 라운드(pill) 다크 바 + 브랜드 마크 + 음성/프로필. (산책온: "코스·지역 검색")
- **카테고리 칩 행:** 네이버의 음식점/카페처럼 **목적 칩**(🌳자연·🛡️안전·🤫조용·🦵관절) 가로 스크롤
- **좌상단 날씨 위젯:** 라운드 카드에 기온 + 미세먼지(목업; 좋은 날 넛지 UX P10과 연결)
- **우측 플로팅 원형 컨트롤(세로 스택):** 레이어, 현위치(파란 마커+방향콘), 나침반/줌
- **하단 바텀시트:** 드래그 핸들 + "지금 주변" 콘텐츠(주변 코스 리스트)
- **현위치 마커:** 네이버식 파란 원 + 헤딩 콘

> 제약: TMAP 기본 타일은 네이버 다크 톤과 다르다(C2). chrome로 네이버 감각을 내되, 타일 100% 일치는 목표 아님. 필요 시 지도 위 다크 오버레이/CSS 필터 실험.

## 7. 이미지 자산 사용 규칙

- `AllTrails-*.jpg` → **레퍼런스 전용**(앱에 노출 금지).

- `navermap.jpg` → 지도 chrome 레퍼런스.
- 로고/일러스트 필요 시 라임그린 + 라운드 폰트로 워드마크 우선.
